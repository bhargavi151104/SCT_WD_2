# Professional Stopwatch

A modern stopwatch web application built using HTML, CSS, and JavaScript.

## Features

- Start / Pause Stopwatch
- Reset Stopwatch
- Lap Times
- Local Storage Support
- Keyboard Shortcuts
- Responsive Design

## Keyboard Shortcuts

- Space → Start/Pause
- L → Add Lap
- R → Reset

## Technologies Used

- HTML5
- CSS3
- JavaScript
